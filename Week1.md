## 1-1. BigQuery 기본지식

~~~
✅ 학습 목표 :
* 데이터 관련 기초 지식(OLTP, SQL, Row, Column, 저장 형태 등)을 설명할 수 있다. 
* BigQuery 관련 기초 지식에 대해서 파악할 수 있다. 
~~~

OLTP: 거래를 하기 위해 쓰이는 데이터베이스 -> 분석을 하기에는 속도가 느려 적절하지 않다.
SQL: 데이터베이스의 데이터를 가져올 때 사용하는 언어
여러 사람의 건강 데이터를 다루고 있을 때,
Row: 데이터 하나를 뜻하며, 각 사람의 데이터가 여기에 해당된다.
Column: 각 데이터의 특징들로, 사람으로 치면 키, 체중, 나이 등등이 여기에 해당된다.
데이터 저장형태: DB 안에 여러 Table이 있고, SQL은 Table에 있는 데이터를 추출하고, 다루는 것이다.



## 2-1. 데이터 활용 Overview

~~~
✅ 학습 목표 :
* 데이터를 활용하는 과정을 설명할 수 있다.
* 데이터를 탐색하는 과정으로 조건과 추출, 요약을 할 수 있다. 
~~~
1. 데이터 분석 목적에 맞게 데이터를 가져오고, 어떤 것을 원하는지 정한다.(지표정의 등 활용)
2. 단일 자료나 서로 연결된 다량의 자료들을 필터링, 추출, 뱐환, 요약할 수 있다.
3. 예상한 것과 실제 결과가 동일한 데이터 결과를 검증해야 한다.
4. 레포트 작성, 피드백 작성 등을 통해 앞의 과정을 수정, 반복한다.


## 2-2. 저장된 데이터 활용하기

~~~
✅ 학습 목표 :
* 데이터가 저장되는 형태를 알고 저장된 데이터를 활용할 수 있다. 
~~~


ERD(Entity Relationship Diagram): 데이터베이스 구조를 한 번에 알아볼 수 있다
-> 어떤 테이블이 있는지, 테이블에 어떤 컬럼들이 있는지, 테이블끼리 어떻게 연결돼 있는지 등을 확인할 수 있다.
Q. ERD가 없다면?
직접 데이터베이스를 확인하면서 필요한 정보들을 확인해봐야 한다. 이후, 활용할 수 있도록 문서로 정리해야 한다.
